# RockBase IoT Wiki

欢迎来到 **RockBase IoT** 开发者文档中心。

## 快速开始

RockBase IoT 专注于为开发者和创客提供高品质的物联网硬件与解决方案。本文档涵盖产品规格、开发指南、项目案例等核心内容。

## 文档结构

| 板块 | 内容 |
|------|------|
| **产品** | 硬件产品规格书与快速入门 |
| **开发者** | 开发指南、API 参考、固件烧录 |
| **项目** | 开源项目与案例教程 |
| **场景** | 典型应用场景与部署方案 |

## 核心产品

- **NM-CYD-C5** — ESP32-C5 核心开发板
- **NM-RF-HAT** — 射频扩展模块
- **NM-Display-2.8** — 2.8 寸触控显示屏
- **NM-EPD-420** — 4.2 寸电子墨水屏

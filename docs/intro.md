# RockBase IoT Wiki

Welcome to the **RockBase IoT** Developer Documentation Center.

## Quick Start

RockBase IoT focuses on providing high-quality IoT hardware and solutions for developers and makers. This documentation covers product specifications, development guides, project cases, and more.

## Documentation Structure

| Section | Content |
|---------|---------|
| **Products** | Hardware product specifications and quick start guides |
| **Developer** | Development guides, API references, firmware flashing |
| **Projects** | Open-source projects and case tutorials |
| **Scenarios** | Typical application scenarios and deployment solutions |

## Core Products

- **NM-CYD-C5** — ESP32-C5 Core Development Board
- **NM-RF-HAT** — RF Expansion Module
- **NM-Display-2.8** — 2.8-inch Touch Display
- **NM-EPD-420** — 4.2-inch E-Paper Display
